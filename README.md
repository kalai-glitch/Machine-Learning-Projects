Heart Disease Prediction using Logistic Regression

Project Overview:
Heart disease is one of the leading causes of death worldwide. This project aims to predict the likelihood of heart disease in a patient based on various medical attributes using Logistic Regression, a supervised machine learning algorithm.
The project demonstrates an end-to-end machine learning workflow, from data preprocessing to model evaluation.


Objectives:
Analyze medical data related to heart disease
Build a prediction model using Logistic Regression
Evaluate model performance using accuracy and other metrics
Predict whether a person is likely to have heart disease
